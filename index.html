About

Welcome to NeoSSG, a webring for Neocities sites made with static site generators ("SSGs") like Jekyll, Hugo, Eleventy, etc. This webring celebrates SSG users of all skill levels and aims to promote diverse and unique personal sites we make using these tools.
Welcome!

Requirements

To keep this webring welcoming to all, websites must meet these requirements:
  • hosted on Neocities
  • built using an SSG (all SSGs are welcome)
